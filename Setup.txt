Setting up your Custom ChatGPT & Midjourney Discord Bot Clone

You're just a few steps away from creating your own AI-powered Discord bot inspired by ChatGPT and Midjourney! This guide will walk you through the setup process to get your bot up and running in no time.

**Before you begin:**
- Make sure you have Python installed on your system.

**Step 1: Get Your API Key**
1. Sign in to https://starsky.pro/developers.
2. Obtain your API key, which will enable your bot to access the Starsky AI capabilities.

**Step 2: Configure Your Bot**
1. Open the `config.json` file.
2. Replace `"bot_token"` with your actual Discord bot token.
3. Replace `"api_key"` with your obtained API key.

**Step 3: Execute the Install Script**
1. Run the `install.sh` script.
2. The script will automatically set up the bot environment and start the bot.

**Step 4: Bot Setup**
1. After running the script, you'll be ready to use the bot. Head to your Discord server.
2. Use the command `$setup` to link your API key with the bot.

**Step 5: Start Creating**
1. Your bot is now active! Start generating text documents and stunning images effortlessly.
2. Use commands like $help or $image and $templates to create content inspired by ChatGPT and Midjourney.
